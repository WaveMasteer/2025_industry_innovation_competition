## 2025_industry_innovation_competition
项目为 "2025年中国大学生工程实践与创新能力大赛" 的 "物流搬运" 赛道的物流车国银级别上下位机开源代码,由于资金有限因此采用了较为平民的打法，因此该项目有较大的上升空间，各位同学可以按照分栏中给出的建议尝试进行二次开发。
受制于本人当时的代码水平，某些地方有对其他佬的代码进行的整合，本身图灵浓度并不足以支撑到国一水平，若在代码中看到缺陷之处请多多谅解…

### STM32F103ZET6
  使用正点原子出品的STM32F103ZET6开发板，没有加装其余外设，与树莓派4B采用UART通讯。
  开发板直接控制四个步进电机、三个270°舵机，其中每个步进电机连接一个张大头v42的步进电机控制驱动，对步进电机有梯形加减速算法，模拟加速度的方式可以减轻步进电机本身扭矩较小无法刚性旋转的缺陷。
